# Cursor Usage

一个用于显示 Cursor 使用情况的 VSCode 扩展。

## 功能

在 VSCode 左侧活动栏显示：
- Premium 使用情况
- Basic 使用情况
- 会员类型
- 试用剩余天数

数据每 20-40 秒自动刷新一次。
